# Coding Test Practice

이 저장소는 Java를 기반으로 다양한 코딩 테스트 문제를 해결하고 정리한 공간입니다. 알고리즘 문제 풀이를 통해 문제 해결 능력을 키우고, 실전 감각을 익히는 것을 목표로 합니다.

## 소개

- 사용 언어: Java
  
- 문제 출처:
  - 프로그래머스
  - 백준
  - PCCP 모의고사
  - 기타 기업 코딩 테스트 예제
    
- 다루는 알고리즘:
  - DFS / BFS
  - DP (동적 프로그래밍)
  - Greedy (탐욕법)
  - 완전 탐색
  - 구현
  - 힙 / 우선순위 큐
  - 문자열 처리
  - 큐 / 스택 등 자료구조
